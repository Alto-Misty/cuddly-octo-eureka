# Лабораторная работа №2: Многопоточная обработка данных

## Условие задания (вариант 20)

Дан массив координат (x, y, z).  
Необходимо найти три точки, которые образуют треугольник максимальной площади.

Программа работает в многопоточном режиме:
- Количество потоков задаётся ключом командной строки.
- Используются стандартные средства ОС для создания потоков (например, `pthreads` в Unix/Linux).
- Используется синхронизация (мьютексы) для доступа к общим данным.
- Проводится исследование производительности (ускорение, эффективность) в зависимости от числа потоков и размера данных.

## Требования:

- Язык: Си (C99 или выше)
- Использовать `pthreads` (POSIX threads) для многопоточности
- Запрещено использовать `stdio.h`, `stdlib.h`, `string.h` (только системные вызовы и `pthread`)
- Ключ запуска: `./program <threads_count> <input_file> <output_file>`
- Входной файл: список координат (x, y, z), по одной точке на строку
- Выходной файл: три координаты и площадь
- Провести исследование: ускорение, эффективность

## Требования к среде:

- Linux / WSL
- gcc
- `libpthread` (обычно входит в `build-essential`)

## Как собрать:

```bash
gcc -o program program.c -lpthread
./program 4 input.txt output.txt
