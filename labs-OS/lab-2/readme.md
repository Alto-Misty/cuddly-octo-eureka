# Лабораторная 2: Максимальная площадь треугольника (многопоточный поиск)

Находит треугольник с максимальной площадью из набора 3D точек, используя `pthread` для распараллеливания.

## Запуск

```bash
# Компиляция
gcc -o lab2 main.c -lpthread -lm

# Запуск
./lab2 <число_потоков> <input.txt> <output.txt>
```

## Тесты

Создайте `input.txt`:
```
0 0 0
1 0 0
0 1 0
0 0 1
```

Запустите:
```bash
./lab2 2 input.txt output.txt
```

Результат в `output.txt`: координаты вершин и площадь лучшего треугольника.
```
